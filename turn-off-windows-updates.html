Desative as atualizações do Windows

Antes da ativação: O PRIMEIRO, O QUE VOCÊ PRECISA FAZER, é certificar-se de que as atualizações do sistema estão DESLIGADAS.


Opção nº 1 altamente recomendada (desligamento manual).

  • Recomendamos desativar manualmente as atualizações (SE não o fizer, a garantia pode ser anulada, não abuse e ignore as instruções)
  • Aqui está pintado em detalhes e mostrado como fazê-lo:
    https://www.techradar.com/how-to/how-to-stop-a-windows-10-update

Veja também o vídeo:

 

Equipes: (pré-pressione Win + R no teclado)

  • services.msc — Abre serviços
  • gpedit.msc — Abre o Editor de Política de Grupo
  • Se você tiver uma versão Home do Windows (ou outra que não tenha o Editor de Política de Grupo) anteriormente Download the Editor at this link (Click) and install (run from the Administrator).

Opção número 2 (desligamento usando o programa)

  • Caso você espere passar rapidamente no jogo, pode desabilitar o Windows Update Service usando Updates Disabler v1.5
  • reference number 1 / número de referência 2
  • — O programa ajudará a desligar rapidamente as atualizações. Além disso, se necessário, as atualizações podem ser ativadas rapidamente por meio dele.
